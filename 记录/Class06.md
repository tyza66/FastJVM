# 深入了解栈

1. 栈是一种数据结构
2. 一般栈跟队列比较
3. 程序=数据结构+算法
4. 栈：先进后出，后进先出，桶
5. 队列：先进先出（FIFO：First Input First Output）
6. 喝多了吐就是栈，吃多了拉就是队列
7. main方法先执行但是最后结束~ 
8. 就是java栈里面运行的时候是一个栈的结构
9. 方法间相互调用就相当于无限压栈，会导致栈溢出问题
10. 栈内存，主管程序的运行，生命周期和线程同步
11. 线程结束，栈内存也就释放了，对于栈来说，不存在内存回收问题
12. 一旦线程结束，栈就没了
13. 栈中放的东西：8大基本类型，对象的引用，实例的方法
14. 栈运行原理：栈以栈帧的形式存在（一片一片的）
